Important: This bot is designed for educational purposes. Always use with caution and never risk more than you can afford to lose.
Getting Started
  1. Create a Deriv Account: If you don't already have one, create an account on Deriv.
  2. Generate an API Token: Log in to your Deriv account, go to API Token, and create a token with the "Admin" scope.
  3. Start the Bot: Enter your API token on the dashboard and click "Start Bot" to begin trading.
  4. Monitor Performance: Watch the bot's performance in real-time on the dashboard.

Customization Options

The bot can be customized by editing the config.py file. Key settings include:

Setting Description
base_stake The default stake amount for trades
max_stake Maximum allowed stake for any trade
kelly_fraction Risk management parameter (0-1) - lower is more conservative
max_daily_loss Maximum allowed loss before the bot stops trading
profit_target Target profit amount for the session
markets List of markets to trade on
FAQ

The bot uses advanced strategies to identify high-probability trading opportunities, but trading binary options always involves risk. Results will vary based on market conditions and the bot's settings. The bot aims to shift the odds more in your favor compared to standard binary options trading, but there are no guarantees of profit.

The bot primarily trades on synthetic indices offered by Deriv, including volatility indices, crash/boom indices, and step indices. These markets are available 24/7 and have predictable volatility characteristics that the bot can analyze effectively.

The bot uses several risk management techniques:
  • Enhanced Kelly Criterion for optimal stake sizing based on win probability
  • Daily loss limits to prevent excessive losses
  • Profit targets to secure gains
  • Modified martingale approach with damping factors for recovery after losses
  • Variable stake adjustments based on market conditions and historical performance

The bot uses a combination of indicators:
  • RSI (Relative Strength Index)
  • MACD (Moving Average Convergence Divergence)
  • Bollinger Bands
  • Price action pattern recognition
  • Trend analysis and strength measurement
  • Volatility analysis
  • Price/indicator divergence detection
These indicators are used together with confirmation logic to generate higher-probability signals.
